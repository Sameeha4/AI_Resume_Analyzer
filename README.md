# 🤖 AI Resume Analyzer

AI Resume Analyzer is a Flask-based web application that uses Generative AI to analyze resumes against job descriptions. It helps users identify matching skills, missing skills, ATS score, and get personalized resume improvement suggestions.

## 🚀 Features

- Upload resume in PDF format
- Analyze resume using AI
- Compare resume with job description
- Generate ATS match score
- Identify matching and missing skills
- Get resume strengths and weaknesses
- AI career counselor for improvement suggestions

## 🛠️ Technologies Used

- Python
- Flask
- Groq API
- pypdf
- HTML
- CSS

## ⚙️ Installation

Clone repository:

git clone https://github.com/Sameeha4/AI_Resume_Analyzer.git

Install dependencies:

pip install -r requirements.txt

Create .env file:

GROQ_API_KEY=your_api_key

Run application:

python app.py

Open:

http://127.0.0.1:5000
🤖 AI Model

Uses Groq API with Llama 3.3 model for resume analysis and career guidance.
